Library
Name

DotEnv
Description

The DotEnv class provides the interface for accessing environment variables including local overrides in a file in the sketch folder.
Methods
get() Returns the value for a given environment variable.
Constructor 
DotEnv(parent)
DotEnv(parent, filename)
Parameters
parent PApplet: typically use "this"
filename String: name of file in the sketch folder that contains variable overrides (defaults to .env if not specified)
Usage Application
